# Tweet-Insta-Search
Given a key term, obtain the 10 most recent tweets.

### Installation, Test, and Tinker

**Steps:**

1. `git clone [project]` and cd into it.
2. Install dependencies ---- `npm i`
3. Run tests ---- `npm test`
4. Tinker with project ---- `npm start`
